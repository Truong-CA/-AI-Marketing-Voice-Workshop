### [Blog 1 - Migrating Data and Reducing Costs at Scale with Amazon S3 File Gateway](3.1-Blog1/)

This blog explains how **Amazon S3 File Gateway** can help organizations migrate large volumes of on-premises data to AWS while preserving the original file structure and creation timestamps. It also demonstrates how retaining metadata enables organizations to take advantage of automated storage tiering and lifecycle policies to optimize storage costs.

###  [Blog 2 - An Overview of Amazon Bedrock AgentCore Payments](3.2-Blog2/)
This blog introduces an AI Agent that can pull out its own wallet to pay for data, call APIs, or even hire another Agent to do work on its behalf — with no human needed to click a confirmation button. That's exactly what AWS just introduced with Amazon Bedrock AgentCore Payments. As more and more services move to pay-per-call pricing costing just fractions of a cent, letting an Agent manage its own wallet and pay autonomously within an authorized budget is no longer a distant vision — it's now a real feature, built on x402, an open protocol developed by Coinbase.

### [Blog 3 - IAM User & IAM Policy: Two Fundamental Concepts When Learning AWS](3.3-Blog3/)

This blog explains why the AWS Root Account should not be used for everyday operations, the different responsibilities of **IAM Users**, **IAM Groups**, and **IAM Roles**, the differences between **Managed Policies** and **Inline Policies**, and how the **Principle of Least Privilege** can be applied through a practical example involving **Intern**, **Developer**, and **Administrator** roles within a small team.
