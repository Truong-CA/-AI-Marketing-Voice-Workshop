Throughout the internship period at **Amazon Web Services Vietnam Co., Ltd.** from **April 20, 2026** to **July 10, 2026**.

During the internship, I participated in developing the AI Marketing Voice on AWS project. The project was carried out by a team of 02 members, in which I was responsible for virtually all technical work, from researching ideas, analyzing requirements, designing the system architecture, building the backend and frontend, integrating AWS services such as Amazon Bedrock, Amazon S3, Amazon CloudFront, Amazon SES, and Amazon CloudWatch, developing the core system features, testing, and finalizing the product. In addition, each member independently built their own architecture diagram and workshop deployment guide as required by the internship program.

Through the project development process, I significantly improved my skills in Python and ReactJS programming, designing and deploying applications on the AWS Cloud platform, integrating generative AI services, RESTful API design, database management, debugging and system testing, problem analysis and resolution, reading technical documentation in English, writing technical documentation, workshops and reports, while also improving my time management, teamwork, and communication skills in a professional working environment.

In terms of work ethic, I always strived to complete tasks well, comply with regulations, and actively communicate with colleagues to improve work efficiency.
