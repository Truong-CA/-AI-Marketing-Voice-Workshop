### Week 1 Objectives

* Get acquainted with the AWS Study Group internship program.
* Gain an overview of the AWS cloud computing platform.
* Create and configure an AWS account.
* Become familiar with AWS Management Console, AWS Support, and AWS Documentation.
* Understand basic AWS service groups in preparation for the learning and project development process.

### Week 1 Results Achieved

* Understood the overall overview of the AWS cloud computing platform and Cloud service models.
* Successfully created an AWS Free Tier account and configured basic security settings.
* Became familiar with the AWS Management Console and learned how to search for and access AWS services.
* Learned how to use AWS Support Center and AWS Documentation to find resources and resolve issues during the learning process.
* Successfully installed and configured AWS CLI on the local machine.
* Executed basic AWS CLI commands to verify account information and the working environment.
* Fully prepared the environment to begin researching AWS services in the coming weeks.
* Understood the AWS global infrastructure architecture, including Regions, Availability Zones (AZs), and Edge Locations, and learned how to select the appropriate Region when deploying resources.

* Gained knowledge of AWS core service groups such as Compute, Storage, Networking, Database, Security, and Management, thereby gaining an overall understanding of the AWS ecosystem.

* Became familiar with the resource management process on AWS, including creating, monitoring, and deleting resources to minimize unnecessary costs.

* Understood the role of Access Key ID, Secret Access Key, and the importance of securing authentication credentials when using AWS CLI.

* Began familiarizing with AWS resource management through both the AWS Management Console and AWS CLI, laying the groundwork for hands-on practice with services in the coming weeks.

* Completed the setup of the AWS learning and practice environment, ensuring readiness to deploy content related to IAM, Amazon S3, Amazon EC2, and other services in the next phase of the internship program.
