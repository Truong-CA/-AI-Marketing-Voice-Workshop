During 12 weeks of internship at AWS Study Group, I gradually explored AWS services, cloud computing architecture, and developed the **AI Marketing Voice on AWS** project. The first six weeks focused on researching AWS services and building foundational knowledge. From week 7 to week 12, I proceeded to design, develop, and complete the project.

| Week | Work Performed | Results Achieved |
|------|----------------|------------------|
| **Week 1** | Explored the overall AWS internship program, became familiar with basic concepts of cloud computing and the AWS ecosystem. Created an AWS account, learned how to use the AWS Management Console, AWS Support, Billing Dashboard, and resource management tools. | Successfully created and configured an AWS account, became familiar with the AWS administration interface, understood basic services, and learned how to use AWS Support and Documentation to assist in the learning and project development process. |
| **Week 2** | Researched AWS networking services including Amazon VPC, Subnet, Route Table, Internet Gateway, NAT Gateway, and Security Group. Practiced building a basic network system on AWS. | Understood AWS network architecture and how components connect to deploy a system. |
| **Week 3** | Studied the IAM service for managing users and access permissions. Researched Amazon S3, how to create Buckets, manage Objects, and configure data access permissions. | Gained understanding of access management and how to use Amazon S3 for secure data storage. |
| **Week 4** | Researched Amazon CloudWatch for system monitoring, log collection, and resource tracking. Also studied Amazon SES for sending emails through AWS. | Understood how to monitor systems using CloudWatch and send automated emails using Amazon SES. |
| **Week 5** | Researched Amazon Bedrock, Generative AI, Foundation Models, and Prompt Engineering. Studied how to integrate the Bedrock Runtime API into Python applications. | Understood the operating principles of Amazon Bedrock and how to build AI-powered applications on AWS. |
| **Week 6** | Analyzed the system requirements for AI Marketing Voice, built the project concept, designed the overall architecture, selected appropriate AWS services, and prepared the development environment. | Completed the system architecture design and project deployment plan. |
| **Week 7** | Initialized the AI Marketing Voice project, built the Backend structure using FastAPI and the Frontend using React, configured the development environment, and connected the database. | Completed the initial project structure and development environment. |
| **Week 8** | Built the marketing content generation feature using Amazon Bedrock, designed the Backend API, and handled Prompt engineering to create advertising content using AI. | Completed the marketing content generation feature using Amazon Bedrock. |
| **Week 9** | Developed the user interface using React, integrated the API between Frontend and Backend, and built user registration, login, and account management features. | Completed the interface and user authentication features. |
| **Week 10** | Integrated the VieNeu-TTS service to convert text to speech. Saved audio files to Amazon S3 and used Amazon CloudFront to distribute audio files to users. | Completed the feature for generating and playing audio files through S3 and CloudFront. |
| **Week 11** | Integrated Amazon SES to send welcome emails when new users register. Configured Amazon CloudWatch Logs to record logs for registration, login, script generation, audio creation, and system error tracking. Conducted testing and application optimization. | Completed supporting features, system monitoring, and improved application stability. |
| **Week 12** | Finalized the AI Marketing Voice project, supplemented Workshop documentation, tested the entire system, optimized source code, prepared the report, and presented the internship results. | Completed the project, Workshop documentation, and final report. |
