### Week 5 Objectives

* Learn about the Serverless Computing model on AWS.
* Get familiar with AWS Lambda and Amazon CloudWatch services.
* Understand how to build event-handling functions and monitor system operations.

### Tasks to Implement This Week

| Day | Task | Start Date | End Date | Resource |
| --- | ---- | ---------- | -------- | -------- |
| Mon | Learn the overview of Serverless Computing and Serverless services on AWS. | 18/05/2026 | 18/05/2026 | https://docs.aws.amazon.com/ |
| Tue | Study AWS Lambda, supported Runtimes, and the deployment process of a Lambda Function. | 19/05/2026 | 19/05/2026 | https://docs.aws.amazon.com/lambda/ |
| Wed | Practice creating a Lambda Function using Python and testing it with sample events. | 20/05/2026 | 20/05/2026 | https://cloudjourney.awsstudygroup.com/ |

### Week 5 Achievements

* Understood the Serverless Computing model and the benefits of deploying applications using a serverless architecture.
* Grasped the operational principles of AWS Lambda and its common use cases.
* Successfully created, deployed, and tested a Lambda Function using the Python language.
* Understood the trigger mechanism of a Lambda Function and the event-handling process on AWS.
* Got familiar with Amazon CloudWatch and its components:
  * CloudWatch Logs
  * CloudWatch Metrics
  * CloudWatch Dashboards
  * CloudWatch Alarms
* Practiced monitoring Lambda Function logs through CloudWatch Logs.
* Learned how to create CloudWatch Alarms to monitor the operational status of AWS resources.
* Understood the role of CloudWatch in monitoring, logging, and troubleshooting applications deployed on AWS.
* Prepared the foundational knowledge to integrate Lambda with API Gateway and other AWS services in the project development phase.
